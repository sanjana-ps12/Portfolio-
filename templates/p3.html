College Bus Management System

This is a web-based project developed to simplify and manage the daily college bus transportation system for students. The system allows students to register, book bus seats, and make fare payments with ease. It helps the admin manage buses, bookings, and student data efficiently.

Key Features:

  • User Registration & Login: Students can register and log in to their personal dashboard.
  • Bus Seat Booking: Allows students to view available buses and book seats in advance.
  • Distance-based Fare Calculation: Automatically calculates travel fare based on the studentâ€™s address.
  • QR Code Boarding System: Generates a QR code to confirm seat booking during boarding.
  • UPI-Based Payment: Students can top up their balance and pay bus fare digitally.
  • Real-time Bus Notifications: Sends updates about bus location and schedule changes.

This project was developed using HTML, CSS (frontend), Flask (Python backend), and MySQL (database). It helps reduce manual work, improve tracking, and enhance the overall student transport experience.
