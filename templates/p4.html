Deepfake Detection System

Python TensorFlow Keras OpenCV Flask

This project focuses on detecting manipulated or AI-generated deepfake videos to combat misinformation and enhance digital media security. It uses advanced Convolutional Neural Networks (CNNs) and frame-level feature extraction to identify subtle irregularities that distinguish real and fake content.

Key Features

  • Video Frame Analysis: Extracts and processes individual frames to detect visual inconsistencies.
  • Deep Learning Model: Uses a CNN-based model trained on real and fake video datasets for high accuracy.
  • Facial Landmark Detection: Identifies abnormal blinking, lighting, and face blending patterns.
  • Web Interface: Built using Flask to upload videos and visualize detection results interactively.
  • Confidence Score Output: Displays the likelihood percentage of a video being fake or real.
