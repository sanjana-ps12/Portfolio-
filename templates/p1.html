Travel Guidance App

Figma UI/UX Design Mobile App

This project is a UI/UX prototype created using Figma for a mobile travel planning application. The app is designed to help users discover travel destinations, explore nearby hotels, and calculate the total cost of a trip ‚Äî all in one place.

Key Features

  • Personalized Travel Recommendations: Curated destination suggestions based on user interests or location.
  • Nearby Hotel Listings: Browse hotels with images, prices, ratings, and amenities.
  • Trip Cost Estimator: Calculate estimated trip cost based on hotel, days, and transport.
  • Online Booking Flow: User-friendly screens for selecting destinations, viewing itineraries, and booking.
